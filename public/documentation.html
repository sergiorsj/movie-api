Documentation for movie-api

Description of Endpoint Return Endpoint URL Method Type Format of Request Data Format of Response Data
Return a list of ALL movies /top10Movies GET None A JSON object holding data about my 10 top movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about the requested movie ex: { "title": "Avatar", "director": "James Cameron", "year": "2009" }
Return data about a single movie's director by title /movies/:title/director GET None A JSON object holding data about the genre of a specific movie ex: "James Cameron"
Return data about a single movie's year by title /movies/:title/year GET None A JSON object holding data about the director of a specific movie ex: "2009"
Return the documentation page /documentation GET None A html page holding the documentation page of this app
Create new user /users POST A JSON object holding data from the user to be added ex: { "fullname": "Jessica Amanda", "email": "jamanda@gmail.com" } A JSON object holding data from the user that has been added plus uuid ex: { "fullname": "Jessica Amanda", "email": "jamanda@gmail.com", "id": "200f45b2-d144-49a6-9252-863c0fec99a9" }
Update user name/email by uuid /users/:id PUT A JSON object holding data from the user to be updated ex: { "fullname": "Leonardo Matheus", "email": "lmatheus@gmail.com" } A JSON object holding data from the user that has been updated ex: { "user": "2", "fullname": "Hilda Mirahi", "email": "hmirahi@gmail.com", "favMovies": [ { "title": "Avatar", "director": "James Cameron", "year": "2009" } ] }
Update users favMovies by title and user id /users/:id/:title POST A JSON object holding data from the user about favMovies to be updated ex: { "director": "James Cameron", "year": "2009" } A message returning if the users favMovies have been successfully updated with the updated favMovies logged on the console ex: [{"title":"The Lion King","director":"Jon Favreau","year":"2019"},{"title":"Fast and Furious","director":"Vin Diesel","year":"2001"}]
Delete a user's favMovie /users/:id/:title DELETE None A message returning if the users favMovies have been successfully updated with the updated favMovies logged on the console ex: [{"title":"Avatar","director":"James Cameron","year":"2009"}]
Delete a user /users/:id/ DELETE None A message returning if the user was successfully deleted
